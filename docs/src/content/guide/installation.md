# Installation

## Package Manager

::: code-group

```bash [pnpm]
pnpm add @lonewolfyx/vue-hooks
```

```bash [npm]
npm install @lonewolfyx/vue-hooks
```

```bash [yarn]
yarn add @lonewolfyx/vue-hooks
```

:::

## Peer Dependencies

`@lonewolfyx/vue-hooks` requires `vue >= 3.3.0` as a peer dependency.

```json
{
    "peerDependencies": {
        "vue": ">=3.3.0"
    }
}
```

Make sure Vue 3 is installed in your project:

```bash
pnpm add vue
```
